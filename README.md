## 📌 Project Title

## 👩‍💻 Author

---

## 📖 Project Overview

This project demonstrates the complete **Procure-to-Pay (P2P)** cycle using SAP MM.

The process includes:

* Purchase Requisition (PR)
* Purchase Order (PO)
* Goods Receipt (GR)
* Invoice Verification
* Payment Processing

---

## 🏢 Business Scenario

A fictional company **Hogwarts Supplies Pvt. Ltd.** manages procurement of magical items like:

* Phoenix Feather Wand
* Polyjuice Potion Ingredients

Vendors include:

* Ollivanders Wand Shop
* Slug & Jiggers Apothecary

---

## 🔄 Process Flow

PR → PO → GR → Invoice → Payment
